# **Customer Cohort Analytics Dashboard**

## **Project Overview**

This project analyzes **customer behavior and retention over time** using cohort analysis on the Metabase internal dataset. Users are grouped by **signup month** to evaluate how different cohorts engage, generate revenue, and retain over time.

---
## **View Dashboard**

[Open Customer Cohort Analytics Dashboard](https://tailed-puffin.metabaseapp.com/public/dashboard/6c5d381f-d8e7-4652-8f3e-eb7dd682c203)
Or check Dashboard PDF file in repo.

---

## **Key Features**

* **Cohort Definition:** Grouped 1,377 users into monthly signup cohorts
* **User Segmentation:** Distribution of users by plan (Basic, Premium, Business)
* **Retention Analysis:** Measured cohort retention and engagement trends
* **Revenue Analysis:** Tracked revenue trends by order date and cohort performance
* **Cohort Size Tracking:** Compared total users across different signup cohorts

---

## **Tools & Technologies**

* SQL (H2 Database)
* Metabase (dashboard & visualization)
* Data aggregation and cohort analysis techniques

---

## **Key Insights**

* Majority of users are on the **Basic plan (~95%)**, indicating potential upsell opportunities
* Revenue trends vary across cohorts, highlighting differences in **customer value over time**
* Cohort size and retention patterns help identify **high-performing and at-risk user groups**
